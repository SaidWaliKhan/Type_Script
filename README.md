## 📖 About

Learning a programming language is best done by building projects.

This repository contains a structured collection of TypeScript projects that gradually increase in complexity—from basic console applications to production-ready backend services. Each project focuses on practical concepts, clean architecture, and modern development practices.

Whether you're a beginner starting your TypeScript journey or a developer looking to strengthen your backend skills, this repository provides a clear roadmap to follow.

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/TypeScript-Projects.git
```

Navigate to the repository:

```bash
cd TypeScript-Projects
```

Choose a project:

```text
Beginner/
    └── Todo-App/
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
